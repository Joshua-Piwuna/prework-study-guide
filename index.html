HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The DOCTYPE decleration is an instruction, and it must be the first line of code in the HTML document
  • Meta datais information about the page that is not displayed directly on the web

CSS

  • CSS stands for Cascading Styling Sheets
  • CSS is used to decorate and style the webpage
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: Used to check what branch we are in and what files have been modified
  • git checkout main: Used to switch to the main branch
  • git pull origin main: This command pulls down the latest main branch

JavaScript

  • JavaScript probides web developers with tools for interactivity, data processing, control and action
